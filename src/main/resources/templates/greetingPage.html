Что бы пользоваться API надо зарегистрироваться или войти на сайт!

Вы вошли на сайт под именем:

Endpoint: /exchange

exchange endpoint возвращает сумму в целевой валюте, после конвертации исходной валюты.

Запрос к API:

http : //localhost:8080/api/ exchange
? from = RUB
 & value = 10000
 & to = USD

Параметры запросса:

Параметр Описание
from Введите трехбуквенный код предпочитаемой базовой валюты.
value Введите объем конвертируемой валюты.
to Введите трехбуквенный код предпочитаемой целевой валюты.

Ответ API:

{ "id" : 26, "result" : 136.474246 }

Объекты ответа:

Объект ответа Описание
id Возвращает id запроса.
result Возвращает объем денежно массы в целевой валюте.

Endpoint: /stats/oneEx

oneEx endpoint предоставляет информацию о пользователях запросивших конвертацию больше {more} $ за один запрос.

Запрос к API:

http : //localhost:8080/api/ stats / oneEx
? more = 100000

Параметры запросса:

Параметр Описание
more Введите целочисленное значение, до какой суммы (в долларах), будет проводиться отбор пользователей.

Ответ API:

{ "users" : [ "Roma","Vasya","Kolya" ] }

Объекты ответа:

Объект ответа Описание
users Возвращает пользователей запросивших конвертацию больше {more} $ за один запрос.

Endpoint: /stats/popCur

popCur endpoint предоставляет рейтинг валют, в которе чаще всего конвертируют пользователи сайта.

Запрос к API:

http : //localhost:8080/api/ stats / popCur

Ответ API:

{ "currency" : { "USD":19,"JPY":4,"RUB":2,"GBP":1 } }

Объекты ответа:

Объект ответа Описание
currency Возвращает рейтинг валют, в которе чаще всего конвертируют пользователи сайта и частоту конвертаций каждой отдельной валюты.
